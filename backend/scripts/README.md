# backend/scripts

Одноразовий dev-інструментарій (не входить у прод-збірку).

## images/ — конвеєр товарних фото
Скрипти, якими збиралися й локалізувалися фото товарів (DuckDuckGo/Amazon/офіційні CDN),
а потім генерувалися карти зображень. Фото вже закомічені в `backend/uploads/products`,
тож у звичайній роботі ці скрипти не потрібні — лишені для відтворюваності конвеєра.

Запуск: `pnpm tsx scripts/images/<script>.ts`
